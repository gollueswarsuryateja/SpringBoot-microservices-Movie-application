3. CONFIGURATION (application.properties)
--------------------------------------------------------------------------------
  movie-service.url  = http://localhost:8081/movies
  rating-service.url = http://localhost:8082/ratings

  NOTE: This service acts as an API Gateway — it proxies requests to:
    - Movie Service  → http://localhost:8081
    - Rating Service → http://localhost:8082

--------------------------------------------------------------------------------
4. APIs
--------------------------------------------------------------------------------

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  ADMIN CONTROLLER  —  Base Path: /api/admin                            │
  ├──────────┬──────────────────┬────────────────────────────────────────── ┤
  │ Method   │ Endpoint         │ Description                               │
  ├──────────┼──────────────────┼───────────────────────────────────────────┤
  │ POST     │ /api/admin       │ Add a new movie                           │
  │          │                  │ Body: Movie { id, name, director, actors }│
  │          │                  │ Proxies to: movie-service.url             │
  ├──────────┼──────────────────┼───────────────────────────────────────────┤
  │ PUT      │ /api/admin/{id}  │ Update an existing movie by ID            │
  │          │                  │ Body: Movie { id, name, director, actors }│
  │          │                  │ Proxies to: movie-service.url/{id}        │
  ├──────────┼──────────────────┼───────────────────────────────────────────┤
  │ GET      │ /api/admin/{id}  │ Fetch movie + its rating by movie ID      │
  │          │                  │ Returns: MovieRating { movie, rating }    │
  │          │                  │ Proxies to: movie-service + rating-service│
  └──────────┴──────────────────┴───────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │  PUBLIC CONTROLLER  —  Base Path: /api/public                          │
  ├──────────┬──────────────────┬───────────────────────────────────────────┤
  │ Method   │ Endpoint         │ Description                               │
  ├──────────┼──────────────────┼───────────────────────────────────────────┤
  │ POST     │ /api/public      │ Submit a rating for a movie               │
  │          │                  │ Body: RatingRequest { movieName, rating } │
  │          │                  │ Proxies to: rating-service.url            │
  └──────────┴──────────────────┴───────────────────────────────────────────┘

--------------------------------------------------------------------------------
5. MODELS
--------------------------------------------------------------------------------

  Movie
  ─────
    - id        : Long
    - name      : String
    - director  : String
    - actors    : List<String>   (@ElementCollection)

  Rating
  ──────
    - id        : Long
    - avgRating : Double
    - name      : String   (movie name)
    - count     : int      (number of ratings)

  RatingRequest
  ─────────────
    - movieName : String
    - rating    : Double

  MovieRating  (Composite response model)
  ───────────
    - movie     : Movie
    - rating    : Rating
